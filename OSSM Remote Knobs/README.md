## OSSM Remote Knobs

![OSSM Remote Knobs](Images/armpit_OSSM_Remote_Knobs.png)  

**Summary**  
Knobs that have the same overall footprint as OSSM default, but have a more minimal design providing a more easily sanitized surface and easier adjustments

- Grooved With Position Indicator  
- Grooved Simple  
- Smooth With Position Indicator  
- Smooth Simple  

**Printing**  
Print with support and with top surface ironing enabled
